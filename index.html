Architecture

GP69 menggunakan pendekatan Asynchronous Multi-Threading. Berbeda dengan ping sitemap biasa, kami memanggil API Google secara langsung untuk memaksa Googlebot mengunjungi URL Anda dalam hitungan menit.

  • Speed: 50-100 URL per detik.
  • Security: Rotasi Service Account otomatis.
  • Stability: Uptime 99.9% di VPS Private Cloud.

Service Account Setup

Untuk privasi maksimal, Anda disarankan menggunakan Service Account (JSON Key) milik sendiri. Berikut langkah singkatnya:

  1. Buka Google Cloud Console.
  2. Create Project > Enable "Indexing API".
  3. Create Service Account > Download JSON Key.
  4. Add email Service Account sebagai Owner di Google Search Console.

Example Request

Berikut contoh payload JSON yang dikirimkan oleh sistem kami ke Google:

POST /v3/urlNotifications:publish
{
  "url": "https://client-domain.com/new-post",
  "type": "URL_UPDATED"
}
